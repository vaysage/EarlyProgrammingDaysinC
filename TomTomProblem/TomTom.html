Problem Statement

 

A boy visits an army camp. In morning he sees a circular row of army guys (call it Rc ) and after some time he observe that they are forming a straight row by splitting this circular at some point( Lets call this straight row R1). In evening he founds that same army guys are in a straight row (call it R2). Boy remembers R1 . Boy wants to know that whether this straight row R2 is again formed by splitting Rc (circular row) at some point or not.


So let’s represent these rows using list of the first character of name of all army man. So you have list R1 and R2 and it is given that R1 is formed by splitting circular list Rc at some point and you have to tell that whether the list R2 is formed by splitting Rc at some place or not.



As shown in above example Boy knows that R1 is formed by splitting Rc (and he remembers R1) but he doesn’t know about R2. In above case it shown that R2 is again made by splitting Rc so for R2 answer is yes but not for R3.

Input/Output Specs

Inputs Specs:
R1(string): List know to be formed by splitting a Circular row 2) R2 (string) - List

Output Specs:
Yes, if R2 is also formed by splitting the circular row at some point. No, Otherwise

 

Examples

Input values:

Example 1 

R1: {C,A,P,G,A,F,J,L,D,H,K,D,M,J}
R2: {K,D,M,J,C,A,P,G,A,F,J,L,D,H}

Example 2

R1: {C,A,P,G,A,F,J,L,D,H,K,D,M,J} 
R2: {P,GK,D,M,J,C,A,A,F,J,L,D,H}


Expected Output value :

Example 1
yes  

Example 2
no
